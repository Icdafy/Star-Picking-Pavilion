# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/)。

## [0.0.2] - 2026-07-23

### Added

- 使用独立于随机本地端口的 `ui-preferences.json` 记住全部有意义的界面选择：主题、最后视图、领域、动态分类、日报日期、常用网址分类与星标，以及实时更新开关。
- 通过版本化白名单、原子写入和损坏回退，使上述选择在完全退出并重新打开桌面应用后仍能恢复。

### Fixed

- 修复 Electron utility process 的 `MessageEvent.data` 凭据确认未被识别，导致 DeepSeek API Key 虽已加密写入却在界面中显示保存失败、运行时无法调用的问题。
- 将 AI 设置更新、加密凭据保存和普通设置落盘串行为完整事务，避免并发更新失败回滚覆盖较新的成功设置。

### Tests

- 新增两次真实 Electron 冷启动回归测试，强制第二次启动使用不同随机端口，并通过本地模拟 DeepSeek 服务验证加密 Key 恢复和模型调用。
- 新增界面偏好存储、IPC 边界、渲染器恢复、凭据确认及设置事务的单元回归测试。

## [0.0.1] - 2026-07-21

### Added

- Windows x64 桌面安装包、自动更新元数据与本地 SQLite 情报库。
- 低空经济、商业航天信息流、全文检索、聚类、日报与信源管理。
- 可选 AI 五维评分、系统加密凭据存储与“云幄 · 常用网址”视图。

### Security

- 随机回环端口、启动令牌、严格 Host/Origin 校验与请求体限制。
- Electron sandbox、权限默认拒绝、CSP、安全外链与渲染崩溃恢复。
- 原子设置写入、SQLite 完整性检查、旧数据一致性迁移与优雅退出。
- 白名单打包、ASAR 审计、生产依赖审计、SBOM 与 SHA-256 发布门禁。

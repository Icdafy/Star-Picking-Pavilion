# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/)。

## [0.0.1] - 2026-07-21

### Added

- Windows x64 桌面安装包、自动更新元数据与本地 SQLite 情报库。
- 低空经济、商业航天信息流、全文检索、聚类、日报与信源管理。
- 可选 AI 五维评分、系统加密凭据存储与“云幄 · 常用网址”视图。

### Security

- 随机回环端口、启动令牌、严格 Host/Origin 校验与请求体限制。
- Electron sandbox、权限默认拒绝、CSP、安全外链与渲染崩溃恢复。
- 原子设置写入、SQLite 完整性检查、旧数据一致性迁移与优雅退出。
- 白名单打包、ASAR 审计、生产依赖审计、SBOM 与 SHA-256 发布门禁。

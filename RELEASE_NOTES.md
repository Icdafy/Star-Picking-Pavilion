# 摘星阁 v0.0.2

本版本重点修复跨冷启动的选择记忆和 DeepSeek 配置保存、调用问题。

## 亮点

- 记住全部有意义的界面选择，包括深浅主题、最后视图、领域与分类、日报日期、常用网址分类与星标、实时更新开关。
- 界面偏好保存在独立的版本化文件中，因此桌面后端每次启动即使使用不同随机端口，也能恢复上一次选择。
- 修复 DeepSeek API Key 无法正常保存和立即使用的问题；加密保存成功后运行时会收到可靠确认，连接测试和后续模型调用可以使用已保存 Key。
- AI 设置保存采用串行事务，避免同时保存时失败回滚覆盖已经成功的新设置。
- 使用两次真实 Electron 冷启动和本地模拟 DeepSeek 服务验证选择恢复、加密凭据恢复与授权模型调用。

## 安全与隐私

API Key 仍使用 Windows 当前用户的系统加密能力保存于 `credentials.v1.json`，不会写入 `settings.json` 或界面偏好文件。`ui-preferences.json` 仅保存经过白名单校验的界面选择，不保存搜索词、滚动位置、未提交表单或其他临时状态。

## 安装提示

本版本未签名。Windows SmartScreen 可能提示未知发布者；请先使用随 Release 提供的 `SHA256SUMS.txt` 核验安装包，再决定是否运行。

系统要求：Windows 10/11 x64。卸载默认保留 `%APPDATA%\摘星阁` 中的数据。

启用 AI 后，文章标题、摘要等待分析文本会发送给用户配置的模型服务。详情见 README 的隐私说明。

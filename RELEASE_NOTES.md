# 摘星阁 v0.0.5

本版本只做一件事：把界面做扎实。渲染层重建了设计令牌体系，修掉几处长期存在但一直没被注意到的布局缺陷，并把此前只能靠鼠标试探的操作补成完整的键盘路径。采集、AI 分析、数据保留、凭据存储与桌面后台运行的行为完全没有改动，升级后所有设置保持原样。

## 亮点

- **导航条真的会停住了。** 此前 `html, body` 的 `height: 100%` 把 sticky 的包含块钉死在视口高度，导航条滚过一屏后就跟着内容滑走。现在它在整篇信息流里稳定悬停，日期分组标题和右侧热度栏也会按导航条的实际高度自动排在它下方，导航换行时跟着变。
- **日期标题不再和卡片叠字。** sticky 的日期分组标题此前没有背景，滚动时下方卡片的文字会直接透上来。现在加了同色薄纱与轻微模糊。
- **五维研判有了入口。** AI 的五维分解此前只能靠点击卡片空白处偶然发现，没有任何可见提示，键盘也到不了。现在是一个明确的展开按钮，带朝向状态和 `aria-expanded`。
- **小字终于看得清。** 此前连按钮、输入框、标签都用仿宋，12px 上笔画细到发虚。现在控件与元信息改用系统黑体，摘要和情报研判仍是仿宋——该有的报馆气质留着，该读的地方读得动。
- **键盘快捷键。** `/` 或 `Ctrl+K` 聚焦检索，`Esc` 清空退出，`Alt+1`–`Alt+7` 切视图，`Alt+T` 换主题，`Alt+R` 立即采集，`Home` 回顶部。设置页有一张速查卡。
- **检索有了上下文。** 检索时顶部显示当前关键词与已列出条数，一键清除；长列表新增回到顶部按钮。
- **对比度达标。** 宣纸白主题下 11–13px 标签的对比度从 4.07:1 提到 4.64:1，满足 WCAG AA；全部可聚焦控件补齐了统一的键盘焦点环。
- **时间轴对齐。** 时间点与竖轴线此前错位约 6.5px，现在共用同一几何基准。

## 升级说明

本版本不改变任何数据结构与配置项，无需迁移。主题、视图、领域、分类、日报日期、常用网址星标与实时开关等界面选择照常从 `%APPDATA%\摘星阁\ui-preferences.json` 恢复。

「桌面运行」的两个开关（关闭到托盘、登录启动）维持你升级前的状态，本版本没有触碰它们。

## 安全与隐私

本版本未改变凭据、网络、AI、数据保留或信源策略。改动全部位于渲染层，且仍受既有内容安全策略约束：页面脚本一律外置，不使用任何内联事件处理器，外链协议白名单与 Electron 的安全外链策略均未放宽。API Key 仍使用 Windows 当前用户的 `safeStorage` 加密能力保存。

## 安装提示

本版本未签名。Windows SmartScreen 可能提示未知发布者；请先使用随 Release 提供的 `SHA256SUMS.txt` 核验安装包，再决定是否运行。

系统要求：Windows 10/11 x64。卸载默认保留 `%APPDATA%\摘星阁` 中的数据。

启用 AI 后，文章标题、摘要等待分析文本会发送给用户配置的模型服务。详情见 README 的隐私说明。

# 摘星阁 v0.0.1

首个 Windows 桌面发行版，统一使用中文名“摘星阁”和英文名“Star-Picking-Pavilion”。

## 亮点

- 聚合低空经济与商业航天信源，支持精选、热点、全文检索、事件聚类和日报。
- 默认可在无 AI Key 的启发式模式运行；也可连接用户自选的 OpenAI 兼容模型。
- AI Key 使用 Windows 系统加密保存，旧数据库通过一致性备份安全迁移。
- 本地 API 使用随机回环端口和启动令牌，渲染器启用 sandbox、CSP 与权限拒绝。
- 安装包经过严格 ASAR 白名单、体积、秘密和生产依赖审计。

## 安装提示

本版本未签名。Windows SmartScreen 可能提示未知发布者；请先使用随 Release 提供的 `SHA256SUMS.txt` 核验安装包，再决定是否运行。

系统要求：Windows 10/11 x64。卸载默认保留 `%APPDATA%\摘星阁` 中的数据。

启用 AI 后，文章标题、摘要等待分析文本会发送给用户配置的模型服务。详情见 README 的隐私说明。

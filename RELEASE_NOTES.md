# 摘星阁 v0.0.4

本版本让“摘星阁是否在后台运行、是否随 Windows 登录启动”完全由用户决定。两个开关都位于「设置 → 桌面运行」，默认关闭且相互独立；升级后不会自动改变现有启动或退出习惯。

## 亮点

- **关闭到托盘由你决定。** 开启“关闭主窗口后在后台运行”后，关闭窗口不会中断采集。可从系统托盘重新打开；要彻底结束应用和本地后端，请在托盘菜单选择“退出摘星阁”。
- **登录启动独立控制。** 正式安装的 Windows 版本可单独开启“登录 Windows 时自动启动”。若同时开启后台运行，登录后静默启动；若只开启登录启动，主窗口会正常显示。
- **失败时保持可恢复。** 若系统托盘无法创建，应用会安全降级为可见窗口并给出固定提示，不会隐藏到用户无法重新打开的状态。
- **界面只显示确认状态。** 两个开关串行提交，主进程或 Windows 回读确认后才更新界面；保存失败时会恢复实际状态。
- **既有安全策略不变。** API Key 加密、随机回环端口、AI 调用、数据保留和信源访问规则均未改变。

## 升级说明

升级后两个桌面运行开关均为关闭状态。后台运行偏好保存在 `%APPDATA%\摘星阁\ui-preferences.json`；Windows 登录启动以操作系统状态为准，不写入该文件。

若开启后台运行，普通的窗口关闭按钮会隐藏窗口。需要退出时，请使用托盘菜单中的“退出摘星阁”；退出仍会等待本地后端完成数据库收尾。再次启动摘星阁或启动第二个实例时，已有窗口会恢复、显示并获得焦点。

## 安全与隐私

桌面设置通过严格白名单 IPC 暴露，渲染页面不能访问通用 Electron 或系统接口。本版本未改变凭据、网络、AI、数据保留或信源策略；API Key 仍使用 Windows 当前用户的 `safeStorage` 加密能力保存。

## 安装提示

本版本未签名。Windows SmartScreen 可能提示未知发布者；请先使用随 Release 提供的 `SHA256SUMS.txt` 核验安装包，再决定是否运行。

系统要求：Windows 10/11 x64。卸载默认保留 `%APPDATA%\摘星阁` 中的数据。

启用 AI 后，文章标题、摘要等待分析文本会发送给用户配置的模型服务。详情见 README 的隐私说明。
